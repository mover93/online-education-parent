# online-education-parent
一、项目介绍
online-education-parent是一套微服务在线教育系统，采用了 Spring Cloud Hoxton &amp; Alibaba、Dubbo、RocketMq、Nacos、Spring Boot 2.3.2、Oauth2、MyBatis、Docker、Elasticsearch等核心技术
